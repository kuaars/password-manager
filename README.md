Менеджер паролей на Python




Описание

Это настольное приложение для управления паролями, написанное на Python с использованием customtkinter. Оно позволяет безопасно хранить пароли, шифруя их с помощью RSA и AES, а также предоставляет функционал генерации и копирования паролей.

Приложение хранит данные в локальной базе SQLite и защищает их мастер-паролем.

Возможности

Хранение паролей с шифрованием RSA + AES.

Мастер-пароль для доступа к данным.

Генератор сложных паролей.

Просмотр, копирование и удаление сохраненных паролей.

Возможность сброса мастер-пароля с удалением всех данных.

Автоматическое создание ключей при первом запуске.

Установка

Клонируйте репозиторий:

git clone https://github.com/yourusername/password-manager.git
cd password-manager


Установите зависимости:

pip install customtkinter pyperclip pycryptodome rsa


Запустите приложение:

python main.py

Использование

При первом запуске введите мастер-пароль (не менее 6 символов).

Ключи RSA будут сгенерированы автоматически и сохранены локально (keys.enc).

После авторизации можно:

Добавлять новые пароли

Генерировать сложные пароли

Просматривать и копировать пароли

Удалять сохраненные пароли

Для сброса мастер-пароля используйте кнопку "Сбросить пароль" на окне авторизации. Все данные будут удалены.

Безопасность

Пароли шифруются с использованием RSA и AES-GCM.

Мастер-пароль не хранится в явном виде.

Данные сохраняются локально в зашифрованном виде (keys.enc и passwords.db).
